# Edit Report

## Step-by-Step Instructions

*Currently, editing reports before sending is a planned feature and is under development.*

In future updates, you will be able to:

1. Generate a report based on filtered tasks.
2. Edit the report content directly within the app.
3. Customize the report format and content as needed.
4. Save and send the customized report via email.

Stay tuned for updates that will include this functionality.
